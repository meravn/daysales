# מבנה קובץ Excel לייבוא משתמשים

הקובץ צריך להכיל את העמודות הבאות (בשורה הראשונה - כותרות):

| שם | נייד | מייל | תעודת זהות | מחוז | תפקיד |
|----|------|------|------------|------|--------|
| ישראל ישראלי | 0501234567 | israel@example.com | 123456789 | מרכז | סוכן |
| רחל לוי | 0521234567 | rachel@example.com | 987654321 | צפון | מנהל מחוז |
| דוד כהן | 0541234567 | david@example.com | 111222333 | | הנהלה |

## ערכי תפקיד מותרים:
- `סוכן` - סוכן ביטוח
- `מנהל מחוז` - מנהל מחוז
- `הנהלה` - הנהלה ראשית

## שמות עמודות אנגלית (אפשרי גם):
name, mobile, email, id_number, district, role (agent/district_manager/management)
